Showtime

The process of clients requesting property showings can be inefficient, with requests coming from various platforms and lacking essential information like the full address and desired time. ShowTime, the premium showing request hub, streamlines the process with a built-in form, ensuring all requests are complete and well-organized. It's ShowTime, baby!

Technologies Used: Python, Django, postgreSQL, HTML, CSS
